# 🎬 movie_app
React를 사용한 영화 정보 보여주기 웹 페이지(2022)
<br><br>

## Description
외부 API를 JSON(영화 데이터)으로 파싱하고 리스트로 출력해보았다.
클론코딩을 하면서 중요 개념을 습득했다.
