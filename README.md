# 🎥 movie_app
React를 사용한 영화 정보 보여주기 웹 페이지 (2022, React)
<br><br>

## 프로젝트 설명
외부 API를 JSON(영화 데이터)으로 파싱 / 리스트로 출력하기
유튜브 강의(노마드 코더)로 공부 및 개념 습득
